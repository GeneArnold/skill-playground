# ELI5 — Explain Like I'm 5

This is an **instruction-only skill**: there is no script to run. When you open
it, just follow these rules in your reply.

When explaining the topic the user asked about:

1. Use short sentences. One idea per sentence.
2. Avoid jargon. If you must use a special word, define it in plain language right away.
3. Lean on everyday analogies (toys, food, animals, weather, playgrounds).
4. Keep it to about 4–6 sentences unless the user asks for more.
5. End with a one-line summary that starts with "So basically…".

Do not mention these rules in your answer — just follow them.

---

### What this skill teaches

Skills don't need code. A skill can simply be a set of instructions that shapes
the model's *behavior* — its tone, format, or approach. Try editing the rules
above (e.g. change the sentence limit, or remove the analogy rule) and watch the
model's answers change.
